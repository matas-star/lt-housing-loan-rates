# Lietuvos būsto paskolų palūkanų normos

Normalizuota oficialaus Lietuvos banko mėnesinė naujų paskolų būstui įsigyti
vidutinių palūkanų normų eilutė. Leidimas `v2026.05` apima 137 nenutrūkstamus
mėnesius nuo 2015-01 iki 2026-05. Naujausia reikšmė – 3,96 % per metus.

## Rodiklis

- Lietuvos PFĮ euro zonos namų ūkių naujų paskolų būstui įsigyti susitarimai eurais;
- vidutinė metinė palūkanų norma;
- laikotarpio vidurkis;
- Lietuvos banko kodas: `PNS.M.A2V.A.C.A.U2.250.200.N.LT.PC___.E.SR`.

Tai nėra individualus banko pasiūlymas, bendra kredito kainos metinė norma
(BKKMN) ar finansinė rekomendacija.

## Failai

- `lb-housing-loan-interest-rates.csv` – normalizuota lentelė;
- `lb-housing-loan-interest-rates.json` – savarankiškai aprašanti JSON API kopija;
- `housing-loan-rates.xlsx` – šiam leidimui gautas oficialus XLSX snapshotas;
- `datapackage.json` – Frictionless Data Package metaduomenys;
- `SHA256SUMS` – leidimo failų kontrolinės sumos.

Gyvas puslapis ir API:

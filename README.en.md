# Lithuania housing-loan interest rates

A normalized monthly series of the Bank of Lithuania's average interest rate
on new euro-denominated housing-loan agreements for households. Release
`v2026.05` contains 137 uninterrupted months from 2015-01 through 2026-05. The
latest value is 3.96% per annum.

The selected Bank of Lithuania series code is
`PNS.M.A2V.A.C.A.U2.250.200.N.LT.PC___.E.SR`. The workbook defines the measure
as an annualised agreed rate / narrowly defined effective rate and a period
average.

This is not an individual bank quote, annual percentage rate of charge (APR),
or financial recommendation.
